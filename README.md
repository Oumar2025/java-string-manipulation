# Phone Contact App

A simple Java console application that stores phone contacts in a text file using the `java.io` package.

## 📌 Features

- Add new contact
- View all contacts
- Search contact by name or phone number
- Delete contact
- Prevent duplicate phone numbers
- Store data permanently in `phone.txt`

## ▶️ How to Run

### 1. Compile the program

```bash
javac PhoneBook.java
```

### 2. Run the program

```bash
java PhoneBook
```
